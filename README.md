# RentalFlow

RentalFlow is a full‑stack rental management application built with **Node.js**, **Express**, **MySQL**, and a **React frontend** deployed on Netlify.  
The backend is hosted on AlwaysData and exposes REST APIs for managing houses, buildings, tenants, and payments.

---

## 🚀 Features
- Manage houses, buildings, tenants, and rental payments
- RESTful API built with Node.js + Express
- MySQL database hosted on AlwaysData
- React frontend deployed on Netlify
- Authentication with stateful sessions
- Responsive UI for desktop and mobile

---

## 🛠️ Tech Stack
- **Backend:** Node.js, Express
- **Database:** MySQL (AlwaysData)
- **Frontend:** React (Netlify)
- **Deployment:** GitHub → AlwaysData (backend), Netlify (frontend)
